## 🏗 Featured Projects

### 🗺 [MemoriesMap](https://github.com/realthejas/memoriesmap)
> Location-based travel memory SaaS with a 3D interactive globe

- Built async media processing pipelines with **BullMQ + Redis** — cut geocoding API calls by ~85% via cache
- **3D Mapbox globe** rendering geotagged memories with custom WebGL overlays
- Subscription billing via **DodoPayments** ($6.99 / $12.99 tiers), full auth and storage on **Cloudflare R2**
- Stack: `Next.js` `tRPC` `PostgreSQL/Neon` `Prisma` `BullMQ` `Redis` `Cloudflare R2`

---

### ☁️ [MOE — Marketplace Aggregator](https://github.com/realthejas/moe-app)
> Serverless marketplace aggregation platform on AWS

- Architected with **AWS CDK + SQS + Lambda** — serverless-first design brings infra cost to near-zero (~$1.50/month at 1k listings, 10k events)
- Webhook receiver Lambda with **HMAC-SHA256 signature verification** + 5-minute timestamp window for replay attack prevention
- Idempotent event processing via **DynamoDB conditional writes** on `eventId` — no duplicates under retry storms
- **Next.js 14** frontend on Vercel + S3, **Clerk**-gated routes, Zod validation, and a real-time activity feed per listing
- Stack: `Next.js` `AWS CDK` `Lambda` `SQS` `DynamoDB` `Clerk` `TypeScript`

---

### 🤖 [Gitflow AI](https://github.com/realthejas/gitflow-ai)
> RAG-based developer knowledge engine for GitHub repositories

- Ingests codebases and generates context-aware documentation using **LangChain + Gemini**
- Semantic search over repo history and PR diffs with vector embeddings
- Stack: `Next.js` `LangChain` `Gemini` `PostgreSQL` `pgvector`

---

### 📝 [Doc Sprint](https://github.com/realthejas/DocSprint)
> Real-time collaborative document editor

- Sub-100ms sync using **Liveblocks** CRDT-backed presence and storage
- Multiplayer cursors, live selections, and conflict-free editing
- Stack: `Next.js` `Liveblocks` `TypeScript` `Tailwind`
